Preview Mode

This is a static preview of the EduBridge UI. To access the full application with authentication and database functionality:

  1. Install dependencies: pip install -r requirements.txt
  2. Run the Flask server: python app.py
  3. Visit http://localhost:5000

Welcome to EduBridge

Connecting students and tutors worldwide to support quality education for all. Inspired by UN Sustainable Development Goal 4: Quality Education.

Find Expert Tutors

Search and connect with qualified tutors across various subjects. Get personalized learning support.

Book Sessions

Schedule study sessions with your chosen tutors at your convenience. Flexible and easy booking system.

Share Knowledge

Register as a tutor and help students achieve their learning goals. Make a difference in education.

Supporting UN SDG 4

EduBridge is committed to ensuring inclusive and equitable quality education and promoting lifelong learning opportunities for all.

Technology Stack

Backend
  • Flask (Python 3.x)
  • Flask-Login (Authentication)
  • bcrypt (Password hashing)
Database
  • Supabase (PostgreSQL)
  • Row Level Security
  • Real-time capabilities
Frontend
  • HTML5, CSS3
  • Bootstrap 5
  • JavaScript
Security
  • Password encryption
  • Session management
  • Role-based access control
